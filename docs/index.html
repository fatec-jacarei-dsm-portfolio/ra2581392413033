2º semestre - NutriTech

Tecnologias: (React, Typescript, Node, PostgreSQL)

3º semestre - Cloud.io

Tecnologias: (React, Typescript, Node, MongoDB)

Descrição do projeto: Cloud.io é um software meteorológico com o objetivo de manter as pessoas informadas sobre condições climáticas extremas. A principal função é o monitoramento do clima e a emissão de alertas para a população sobre fortes ventos na região, garantindo mais segurança e antecedência para evitar situações de risco.

Atuação: Criação de menus, telas e o sistema de alertas


4º semestre - LabTemp

Tecnologias: (React Native, Typescript, Node, PostgreSQL, C++)

Descrição do projeto: O projeto tem como objetivo fazer um aplicativo multiplataforma, para o acompanhamento e monitoração de temperaturas em estufas utilizando um sensor.

Atuação: Código do microcontrolador, envios dos dados do sensor para o banco de dados/dashboard e tela de home do aplicativo
